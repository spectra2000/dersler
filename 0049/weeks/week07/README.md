<h2>07. Grafikler ve Veri Görselleştirme</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler, Streamlit kullanarak verileri görsel hale getirmeyi öğreneceklerdir.
Grafiklerin kullanıcı arayüzlerinde nasıl etkili biçimde sunulacağı ve farklı grafik türlerinin
hangi senaryolarda tercih edilmesi gerektiği ele alınacaktır.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>Veri görselleştirme temel prensipleri ders notları</li>
    <li>Streamlit grafik bileşenleri dokümantasyonu</li>
    <li>Matplotlib ve Plotly kullanım özetleri</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>Streamlit ile grafik oluşturma anlatım videosu</li>
    <li>Plotly etkileşimli grafik demo videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>Streamlit Chart API Referansı</li>
    <li>Matplotlib Resmi Dokümantasyonu</li>
    <li>Plotly Python Dokümantasyonu</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li>CSV dosyasından veri okuma</li>
    <li><code>st.line_chart()</code> ile hızlı grafik oluşturma</li>
    <li>Matplotlib kullanarak özelleştirilmiş grafik çizimi</li>
    <li>Plotly ile etkileşimli grafikler üretme</li>
    <li>
        Kullanılan araçlar ve kütüphaneler:
        Python, Streamlit, Pandas, Matplotlib, Plotly
    </li>
    <li>
        Ele alınan grafik türleri:
        Çizgi grafikleri, çubuk grafikleri, dağılım grafikleri,
        pasta grafikleri ve etkileşimli grafikler
    </li>
</ul>
