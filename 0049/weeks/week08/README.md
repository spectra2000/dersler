<h2>08. Mini Uygulama – Streamlit ile Paket Program Geliştirme</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler, önceki haftalarda öğrendikleri Streamlit bileşenlerini bir araya
getirerek küçük ölçekli ama çalışabilir bir paket program geliştireceklerdir.
Amaç, tek bir uygulama üzerinden arayüz, veri okuma, grafik ve kullanıcı etkileşimini
birlikte kullanabilmektir.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>Streamlit mini proje geliştirme rehberi</li>
    <li>Örnek Streamlit uygulama senaryoları</li>
    <li>Ders Slaytları</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>Mini paket program geliştirme anlatım videosu</li>
    <li>Streamlit mini proje demo videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>Streamlit Resmi Dokümantasyonu</li>
    <li>Streamlit Uygulama Örnekleri</li>
    <li>Python Veri İşleme ve Görselleştirme Kaynakları</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li>Mini proje senaryosunun belirlenmesi</li>
    <li>Streamlit bileşenlerinin birlikte kullanılması</li>
    <li>Kullanıcıdan veri alma (dosya veya form)</li>
    <li>Verinin tablo olarak gösterilmesi</li>
    <li>En az bir grafik ile verinin görselleştirilmesi</li>
    <li>Kullanıcı etkileşimine göre arayüzün güncellenmesi</li>
    <li>
        Kullanılan araçlar ve kütüphaneler:
        Python, Streamlit, Pandas, Matplotlib / Plotly
    </li>
</ul>
