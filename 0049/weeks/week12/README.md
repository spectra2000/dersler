<h2>12. Paketleme ve Proje Yapısı</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler, geliştirdikleri Streamlit tabanlı uygulamaları düzenli,
okunabilir ve dağıtıma hazır hale getirmeyi öğreneceklerdir. Amaç, bir paket
programın yalnızca çalışmasının değil, aynı zamanda sürdürülebilir ve paylaşılabilir
olmasının önemini kavramaktır.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>Proje klasör yapısı ve modülerlik ders notları</li>
    <li>Python bağımlılık yönetimi dokümantasyonu</li>
    <li>Ders Slaytları</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>Paket program proje yapısı anlatım videosu</li>
    <li>Streamlit proje organizasyonu demo videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>Python proje yapısı best practices</li>
    <li><code>requirements.txt</code> kullanımı</li>
    <li>Virtual environment (venv) rehberleri</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li>Proje klasör yapısının oluşturulması</li>
    <li>Dosya ve modül düzeninin yapılması</li>
    <li>Bağımlılıkların <code>requirements.txt</code> ile tanımlanması</li>
    <li>Virtual environment (venv) kullanımı</li>
    <li>
        Paket programlar için konfigürasyon kavramının uygulanması
    </li>
    <li>
        Kullanılan araçlar:
        Python, Streamlit, pip, <code>requirements.txt</code>, venv
    </li>
</ul>
