<h2>11. AI / LLM Entegrasyonu ile Paket Program Geliştirme</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler, yapay zekâ ve büyük dil modellerini (LLM) paket programlara
entegre etmeyi öğreneceklerdir. Amaç, Streamlit tabanlı uygulamalarda AI servislerini
kullanarak daha akıllı ve etkileşimli programlar geliştirmektir.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>Yapay zekâ ve LLM kavramları ders notları</li>
    <li>AI / LLM API kullanımına giriş dokümanı</li>
    <li>Ders Slaytları</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>AI ve LLM sistemlerinin çalışma mantığını anlatan video</li>
    <li>Streamlit ile AI entegrasyonu demo videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>AI / LLM API servisleri genel dokümantasyonu</li>
    <li>JSON veri yapısı ve API iletişimi</li>
    <li>API anahtarı ve gizli bilgi yönetimi rehberleri</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li><code>st.text_area()</code> ile kullanıcıdan metin alma</li>
    <li>API anahtarı kullanımı ve gizli bilgilerin yönetilmesi</li>
    <li>AI / LLM servisinden cevap alma</li>
    <li>AI çıktısının Streamlit arayüzünde gösterilmesi</li>
    <li>
        Basit hata yönetimi içeren,
        AI destekli bir Streamlit paket programı geliştirme
    </li>
    <li>
        Kullanılan araçlar ve teknolojiler:
        Python, Streamlit, <code>requests</code>, AI / LLM API servisleri
    </li>
</ul>

<h3>Ödev</h3>
<ul>
    <li>
        <strong>Ödev 11:</strong>

        AI tarafından üretilen cevabı arayüzde göstermelidir.
    </li>
</ul>
