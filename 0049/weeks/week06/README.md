<h2>06. Dosya İşlemleri ve Veri Okuma</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler Streamlit tabanlı paket programlarda
dosya okuma, yazma ve kullanıcıdan alınan dosyalarla çalışma
mantığını öğreneceklerdir. Gerçek veri kullanan uygulamaların
temeli bu hafta atılacaktır.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>Streamlit File Uploader Dokümantasyonu</li>
    <li>Python Dosya İşlemleri Özeti</li>
    <li>Örnek Veri Dosyaları</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>Streamlit ile Dosya Yükleme Videosu</li>
    <li>CSV Dosyası Okuma Demo Videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>Streamlit <code>st.file_uploader</code> API Referansı</li>
    <li>Python File Handling</li>
    <li>CSV Veri Yapısı Nedir?</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li><code>st.file_uploader</code> kullanımı</li>
    <li>Yüklenen dosyanın türünü kontrol etme</li>
    <li>Metin dosyasının ekranda gösterilmesi</li>
    <li>CSV dosyasının tablo olarak gösterilmesi</li>
    <li>Basit veri ön izleme uygulaması geliştirme</li>
    <li>
        Kullanılan temel bileşenler:
        <code>st.file_uploader</code>,
        <code>st.write</code>,
        <code>st.dataframe</code>,
        <code>pandas.read_csv</code>
    </li>
</ul>

    </li>
