<h3>Dersin Amacı</h3>
<p>

<h3>Uygulamalar</h3>
<ul>
    <li><code>st.text_input</code> kullanımı</li>
    <li><code>st.text_area</code> ile çok satırlı veri girişi</li>
    <li><code>st.selectbox</code> ve <code>st.radio</code> kullanımı</li>
    <li><code>st.slider</code> ile sayısal değer alma</li>
    <li><code>st.file_uploader</code> ile dosya yükleme</li>
    <li>
        Kullanıcıdan farklı türlerde veri alan
        etkileşimli bir Streamlit paket programı geliştirilmesi
    </li>
    <li>
        Kullanılan temel bileşenler:
        <code>st.text_input</code>,
        <code>st.selectbox</code>,
        <code>st.slider</code>,
        <code>st.file_uploader</code>,
        <code>st.write</code>
    </li>
</ul>
