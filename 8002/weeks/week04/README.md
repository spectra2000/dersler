<h2>04. HTML Temelleri</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftanın sonunda öğrenciler; HTML’in ne olduğunu ve ne işe yaradığını anlayacak,
bir web sayfasının temel yapısını öğrenecek,
en sık kullanılan HTML etiketlerini tanıyacak
ve basit bir HTML sayfası oluşturabilecektir.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>HTML temel kavramları ders notları</li>
    <li>HTML belge yapısı örnekleri</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>HTML belge yapısını anlatan şematik görseller</li>
    <li>Basit HTML sayfası oluşturma videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>HTML (HyperText Markup Language) hakkında temel okuma materyalleri</li>
    <li>HTML, CSS ve JavaScript arasındaki farklar</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li>Basit bir HTML dosyasının oluşturulması</li>
    <li>Temel HTML etiketlerinin kullanımı (html, head, body, h1, p)</li>
    <li>Oluşturulan HTML dosyasının tarayıcıda görüntülenmesi</li>
</ul>

<h3>Ödev</h3>
<ul>
    <li>
        <strong>Ödev 4:</strong>
        Temel HTML etiketlerini kullanarak başlık ve paragraf içeren
        basit bir web sayfası oluşturunuz.
    </li>
</ul>
